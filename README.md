# Git 仓库扫描器

一个轻量级批处理脚本，帮助你在 Windows 上查找所有本地 Git 仓库。

## 🌟 功能
- 扫描任意盘符（D:\, C:\ 等）
- 显示仓库路径、远程地址和最近提交
- 无需安装，双击即可运行
- 兼容中文 Windows（使用 ANSI 编码）

## 💻 使用方法
1. 下载 `scan_git_repos.bat`
2. 修改 `ROOT_PATH` 变量（如 `C:\Users\` 或 `D:\`）
3. 双击运行
4. 在命令行中查看结果
